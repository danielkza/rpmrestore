dependances

  • perl

package rpm

Comme le logiciel travaille sur les rpm, il est naturellement fourni sous ce format. Le package ne dépends donc pas de l'os (type : noarch).
l'installation se fait par la commande rpm -Uvh rpmrestore*.noarch.rpm

tar.gz

pour les besoins éventuels (?) de dépannage, le code est également fourni au format tar.gz.
L'installation se fait par :
  1. tar xvfz rpmrestore-{version}.tar.gz
  2. cd rpmrestore*
  3. make
  4. make install

ps : la commande rpmbuild -tb rpmrestore*.tar.gz permet de fabriquer un package rpm à partir du tar.gz
